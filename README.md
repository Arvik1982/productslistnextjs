# DummyJSON Shop

Next.js приложение интернет-магазина с использованием DummyJSON API. Реализована авторизация пользователей и отображение списка товаров.

## 🚀 Технологии

- **Next.js 16** (App Router)
- **TypeScript**
- **Zustand** (управление состоянием)
- **Axios** (HTTP-запросы)
- **SCSS Modules** (стилизация)
- **JWT** (авторизация)
- **Docker** (контейнеризация)

## 📋 Функционал

- ✅ Авторизация пользователей с JWT
- ✅ Валидация формы входа
- ✅ Отображение списка товаров (12 карточек)
- ✅ Карточка товара с изображением, названием, категорией, ценой
- ✅ Кнопка "Add to cart" только для авторизованных пользователей
- ✅ Header с именем пользователя и кнопкой Logout
- ✅ Footer с текущим годом и email пользователя
- ✅ Адаптивный дизайн
- ✅ Индикаторы загрузки
- ✅ Обработка ошибок API
